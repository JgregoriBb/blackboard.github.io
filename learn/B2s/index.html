Blackboard Building Blocks

Building Blocks are Blackboard Learn's legacy integrations. Written in Java, packaged as a Tomcat Webapp, and installed directly to Learn to run in-process, these integrations have been around for almost 2 decades. This section is designed to help you, the developer, troubleshoot existing Building Blocks.

NOTE: if you are building a new integration, we recommend using Blackboard Learn's REST APIs and IMS Global's Learning Tools Interoperability (LTI).
