SOAP Web Services

SOAP Web Services are one of Blackboard Learn's legacy integrations, and have been deprecated in favor of the more modern REST APIs. This section is designed to help you, the developer, migrate existing Web Service integrations to REST.

NOTE: These are deprecated and will be removed from a release in the very near future. You should use Blackboard Learn's REST APIs and IMS Global's Learning Tools Interoperability (LTI).
